### SECTION 2: HOW IT WORKS

**ラベル**: How It Works（DM Sans, 11px, uppercase, gold, letter-spacing: 4px）
**見出し**: 強みを集める。強みで選ぶ。強みを育てる。

3カラム横並び（モバイルは縦積み）。間に「→」矢印。

| STEP | 見出し | 主語ラベル | コピー |
|------|--------|-----------|--------|
| 1 | 強みを集める | — プロが主語 | セッション後、クライアントがQRコードかNFCカードから投票。「結果を出してくれた」「説明がわかりやすかった」「空間が心地よかった」── あなたの強みを、クライアントの本音が教えてくれる。 |
| 2 | 強みで選ぶ | — クライアントが主語 | 「姿勢改善が得意な人」「メンタルケアに強い人」── クライアントは★ではなく、強みでプロを選ぶ。広告費を払う必要はない。あなたの強みが、集客になる。 |
| 3 | 強みを育てる | — プロが主語 | 投票は蓄積される。辞めても、独立しても、消えない。続けるほどあなただけの「強みの証明」が育っていく。資格では見えない成長が、クライアントの声で見える。最初の1票から始まる。 |

ステップ番号: 丸囲み数字（border: 1.5px solid gold, border-radius: 50%）

### SECTION 3: 比較テーブル

| | ホットペッパー | Google | REAL PROOF |
|---|---|---|---|
| 誰が評価 | 匿名の誰か | 匿名の誰か | 認証済みクライアント |
| 紐づき | 店舗のみ | 店舗のみ | 個人＋店舗 |
| 評価の軸 | ★5（一次元） | ★5（一次元） | 多次元（結果×人柄） |
| 検索順 | 広告費順 | アルゴリズム | 強み順 |
| 不正対策 | なし | なし | 対面認証（QR+NFC） |
| 独立時 | リセット | リセット | 持ち運べる |

### SECTION 4: 3本柱

**見出し**: REAL PROOFが他と違う、3つの理由。

3枚のカード横並び（白背景、1px border、hover時ゴールドborder + 影 + translateY）

| ラベル | 見出し | サブコピー |
|--------|--------|-----------|
| 蓄積 | 辞めても、消えない。 | あなたの実績は店ではなく個人に紐づく。独立しても、転職しても、あなたの強みの証明は一生あなたのもの。 |
| 多次元 | ★じゃ、わからない。 | 「★4.5」で何がわかる？ REAL PROOFは「痛み改善が得意」「メンタルに強い」── 本当の強みが見える。 |
| 信頼性 | 嘘が、つけない。 | 投票できるのは対面でサービスを受けた人だけ。匿名だから忖度もない。届くのは本音だけ。 |

ラベル: ゴールド、12px、letter-spacing: 3px

## 5. インタラクション

| 要素 | 挙動 |
|------|------|
| ヒーロー | fadeUp順次アニメーション（各要素0.15s遅延） |
| 各セクション | IntersectionObserver でスクロール時fadeUp（threshold: 0.12） |
| カード（3本柱） | hover: border-color gold + box-shadow + translateY(-3px) |
| ボタン | hover: translateY(-1px) + box-shadow |
| ナビ | position: fixed, backdrop-filter: blur なし（Dark背景なので不要） |
